## Input contract

Prepare two `.npy` arrays:

```text
points  float32 [1,2048,4]       normalized XYZ + constant weld-category feature
adj     float32 [1,2048,2048]    k=6 adjacency generated by validated preprocessing
```

The runner does not resample points, construct adjacency, relax shapes, convert dtypes, rebuild an engine, or fall back to PyTorch.

## Run inference

```powershell
E:\GRP-PTv2\.venv_ptv2\Scripts\python.exe `
  E:\GRP-PTv2\scripts\run_gcn_res_tensorrt_production.py `
  --manifest E:\GRP-PTv2\artifacts\gcn_res_tensorrt\20260717_173128_144483_phase8d_production_baseline\package\manifests\deployment_manifest.json `
  --points <points.npy> `
  --adj <adj.npy> `
  --output <logits.npy>
```

Output:

```text
logits float32 [1,2048,2]
```

A sibling `<logits.npy>.json` records deployment ID, hashes, input/output information, runtime status, plugin instance count, and ErrorRecorder count.

Before inference, the entry point validates:

- production manifest status;
- Engine, Plugin, and ONNX presence and SHA-256;
- exact TensorRT version and compute capability;
- points/adj shape, dtype, and finite values.

Any mismatch exits nonzero before enqueue. There is no implicit alternate engine, baseline plugin, PyTorch fallback, engine build, or shape relaxation.

## Known numerical exception

The baseline is task-equivalent, not strictly numerically equivalent:

- 18/18 labels match PyTorch and the previous TensorRT baseline;
- all task metrics are identical;
- 13/18 samples satisfy per-sample `max_abs < 1e-4`;
- worst sample `weld_14`: `1.2302398681640625e-4`;
- deterministic status: `DETERMINISTIC_LABELS_ONLY`.

Do not change this tolerance or describe the package as strictly numerically equivalent.

## Manual rollback

The previous validated baseline remains untouched. Its hashes are recorded in `package/manifests/rollback_manifest.json`.

Rollback requires an explicit manifest:

```powershell
E:\GRP-PTv2\.venv_ptv2\Scripts\python.exe `
  E:\GRP-PTv2\scripts\select_gcn_res_tensorrt_baseline.py `
  --manifest <validated-baseline-manifest.json>
```

The selector only updates `deployment/tensorrt/current_baseline.json`. It does not copy, rename, remove, or overwrite engine files. Automatic rollback is prohibited.

## Operational boundaries

This package is not direct robot or safety-controller integration. Downstream safety thresholds, post-processing, label semantics, data sampling, and fallback policy remain outside this runtime and must not be silently changed.
